# PipeRider Report

## Installation

```sh
$ npm install
```

## Development

### Single Report

```sh
$ npm run start:single
```

### Comparsion Report

```sh
$ npm run start:compare
```

## Build

> Generated **single report** and **comparison report** will place into `piperider_cli/data/report`

### Single Report

```sh
$ npm run build:single
```

### Comparsion Report

```sh
$ npm run build:compare
```
