# PipeRider Report

## Installation

```sh
$ npm install
```

## Development

For the single report:

```sh
$ npm run start:single
```

For the comparison report:

```sh
$ npm run start:compare
```

## Build

For the single report:

```sh
$ npm run build:single
```

For the comparsion report:

```sh
$ npm run build:compare
```
